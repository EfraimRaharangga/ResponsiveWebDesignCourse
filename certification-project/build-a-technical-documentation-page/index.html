Introduction

PHP (Hypertext Preprocessor) is a widely-used, open-source server-side scripting language designed specifically for web development. Unlike client-side languages like HTML or JavaScript, PHP code is executed on the server, generating HTML which is then sent to the client's browser. Here's the key features of PHP

  • Server-Side: Runs on the server, keeping your source code hidden from users.
  • Database Integration: Seamlessly connects with MySQL, PostgreSQL, Oracle, and more.
  • Large Ecosystem: Powers platforms like WordPress, Wikipedia, and Laravel.
Requirement

Before you can run PHP, you need an environment that can process it. and here's what you need:

  • Web Server: Apache or Nginx are the most common choices.
  • PHP Engine: The latest stable version (currently PHP 8.x).
  • Database: MySQL or MariaDB is the standard pairing.
Basic Syntax

Every PHP script begins and ends with specific tags. This tells the server to start interpreting the code as PHP. Here's the example of simple php's code

<?
php echo "Hello, World!";
?>

Key Rules

  • Each statement must end with a semicolon (;).
  • Variables start with a dollar sign ($), such as $myVariable.
  • PHP is case-sensitive for variables but not for built-in functions (though it's best practice to be consistent).
PHP Lifecycle

Understanding how a PHP script executes is fundamental to debugging and optimization. When a request hits the server, the PHP interpreter starts at the top of the file and moves downward. It ignores anything outside of the <?php ... ?> tags, passing it directly to the output. This allows you to mix PHP logic with HTML seamlessly.

This "stateless" nature is a core concept. Every time a page is refreshed, the script starts from scratch. It doesn't "remember" what happened in the previous request unless you specifically use sessions, cookies, or a database to persist data.

Basic Syntax Example

<?php
// This is a PHP comment
$greeting = "Welcome to my site";
echo "<h1 >" . $greeting . "</h1 >";
?>
Variables and Data Types

Variables in PHP are the primary way we store and manipulate data. Every variable starts with a dollar sign ($), followed by a name. One of PHP's most flexible features is that it is "loosely typed." This means you don't have to explicitly tell PHP that a variable will hold a string or a number; the engine determines the type automatically.

Primary Data Types:

  • String: Textual data like "Hello World".
  • Integer: Whole numbers like
  • 100. Float: Decimal numbers like 10.5.
  • Boolean: Conditional values, either true or false.
  • Null: A variable with no value assigned.
Constants

While variables are designed to change, constants are used for values that should remain the same throughout the entire execution of your script. You define them using the define() function or the const keyword. Common uses include storing database credentials or site-wide configuration settings. Once defined, they cannot be changed or "unset."

Predefined "Magic" Constants: PHP provides special constants that change based on where they are used:

  • __LINE__: The current line number in the file.
  • __FILE__: The full path and filename of the file.
  • __DIR__: The directory of the file.
Conditional Logic

Conditional statements are the "brain" of your application. They allow your code to take different paths based on specific criteria. The if...else structure is the most common, checking if a condition is true before executing code.

Common Structures

  • If/Else: Standard binary decision making.
  • Switch: Efficient for comparing one variable against many values.
  • Match (PHP 8+): A modern, more concise version of switch that returns a value.
$status = 200; $message = match($status) {
200 => "Success",
404 => "Not Found",
default => "Unknown Error",
};
Loops

Loops execute the same block of code repeatedly until a condition is met. This is essential for tasks like displaying products or reading lines from a file.

  • For Loop: Used when you know the exact number of iterations.
  • While Loop: Runs as long as a certain condition remains true.
  • Foreach Loop: Specifically designed to walk through every item in an array without needing a counter.
$colors = ["Red", "Green", "Blue"];
foreach ($colors as $value) {
echo "Color: $value <br>";
}
Functions

A function is a reusable block of code. Instead of writing the same logic ten times, you wrap it in a function and "call" it whenever needed. This follows the DRY (Don't Repeat Yourself) principle. In modern PHP, you can also use "Type Hinting" to ensure functions receive the correct data types.

Defining a Functions :

function calculateTotal(int $price, int $tax): int {
return $price + $tax;
}
echo calculateTotal(100, 10); // Outputs 110
